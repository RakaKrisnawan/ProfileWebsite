# Personal Profile Website

Website ini dibuat oleh **Made Raka Krisnawan Maheswara** sebagai proyek pembelajaran di ITB STIKOM BALI RENON.  
Tujuan website ini adalah untuk menampilkan profil pribadi, portofolio, serta informasi kontak.

---

## Fitur
- **Tentang Saya**: Deskripsi singkat mengenai latar belakang dan minat.
- **Portofolio**: Daftar proyek yang pernah dibuat, termasuk Todo List App dan Profile Website.
- **Sedang Dipelajari**: Daftar materi yang sedang dipelajari (HTML, CSS, Database, Java).
- **Pendidikan**: Informasi kampus dan jurusan.
- **Kontak**: Email, GitHub, dan LinkedIn.

---

## Teknologi
- HTML5
- CSS3
- (Opsional) PHP & MySQL untuk proyek backend
